Discordance

Opinionated, contradiction-aware knowledge graph for olfactory receptors in cancer — Person C pitch board (5 minutes).

01 · Problem

AI scientists merge. Biologists need disagreement.

K Pro aggregates evidence well. It is not built to notice when primary literature disagrees — especially in niches Owkin has no infrastructure for, like olfactory receptors in cancer.

02 · What we built

MCP tools over a deep, verified OR graph

  • Contradiction detection (don’t silent-merge)
  • Evidence weighting by source strength
  • Confidence-qualified rules
  • Elicitation when masses deadlock

Scope: OR51E2 first, fully verified — not the whole OR family.

04 · Why MCP

Only a live tool can stop mid-reasoning

A static report always invents a conclusion. Discordance pauses on deadlock and elicits:

  • Prioritize proliferation endpoint?
  • Prioritize invasiveness/growth?
  • Keep contested?

06 · Backup if challenged

Don’t freeze — reframe
